# BridgeSprint Dashboard

BridgeSprint is a frontend dashboard designed for **Business Analysts, Scrum Masters, and Project Managers** working with implementation teams.

It centralizes developer updates, sprint metrics, offshore communication, and client updates so implementation teams can easily provide progress updates in client meetings.

## Features

• Sprint progress visualization
• Sprint burndown chart
• Developer workload analytics
• Team status updates
• Drag-and-drop Kanban board
• Client availability tracker
• Offshore team updates
• Stand-up report generator
• Dark mode dashboard
• Search for stories
• Export stand-up report as PDF

## Project Purpose

Implementation teams working for consulting companies often cannot join client meetings directly. Scrum Masters and Business Analysts must collect updates from engineers and present them to clients.

BridgeSprint provides a centralized dashboard where:

• Developers update their work
• Scrum Masters generate stand-up reports
• Teams track sprint progress
• Offshore updates are visible instantly
